# MVP apimtis

Šis dokumentas apibendrina Proof-of-Concept funkcionalumą vienai mokyklai.

## Pagrindiniai vaidmenys
- ADMIN, TEACHER, STUDENT.
- Naudotojus kuria ADMIN per Django Admin.

## Taškų logika
- Vykdomas aktyvus semestras.
- Mokytojai turi semestro biudžetą ir gali skirti taškus.
- Mokiniai gali išpirkti bonusus.

## Puslapiai
- /login/ ir /logout/.
- Mokytojo skydelis, taškų skyrimas, reitingas.
- Mokinų skydelis, bonusų parduotuvė.

## Pastabos
- Architektūros schema bus pridėta į `docs/architecture.*` ir bus laikoma šaltiniu diegimo sprendimams.
